## 🚀 Recent Work (Updated June 17, 2026)

- On **June 16, 2026**, I turned the *Unknown Words* popup in my Spanish-vocabulary browser extension [contexto](https://github.com/Jason-Latz/contexto) into a real review surface in [PR #1](https://github.com/Jason-Latz/contexto/pull/1): Spanish-first word chips with hover/focus reveal, a soft "mark as known," and an SM-2 flashcard practice mode over your stalest saved words — plus a storage-durability fix so concurrent tabs can no longer clobber each other's lexicon writes (18 new tests, 54 passing).
- Also on **June 16, 2026**, I got my on-device Spanish tutor app **Yapper** 🔒 TestFlight-submission-ready (everything short of the credentialed Apple upload): [PR #6](https://github.com/Jason-Latz/yapper/pull/6) made the speech recognizer a pluggable backend — shipping on Apple's on-device STT with NVIDIA Parakeet benchmarked, documented, and deferred behind the same seam — and [PR #7](https://github.com/Jason-Latz/yapper/pull/7) forced fully on-device recognition so the "100% on device" privacy claim is actually true, hardened the app against mid-session audio crashes and call/Siri interruptions, and wired up the App Store submission config.
- Across **June 13–14, 2026**, I shipped a dense run on my journaling app **Vinyl** 🔒: [PR #17](https://github.com/Jason-Latz/vinyl/pull/17) added speak-the-date entry dating, customizable reminders, and the warm paper-and-brass design language (plus a code-rendered app icon); [PR #18](https://github.com/Jason-Latz/vinyl/pull/18) was a correctness / performance / accessibility optimization pass (Insights aggregation moved into SQL, a WCAG-AA palette); and [PR #19](https://github.com/Jason-Latz/vinyl/pull/19) recycled the Library list for flat memory and slimmed the full-text search index — growing coverage from 66 to 115 tests along the way.
- On **June 11, 2026**, I relaunched my personal site with ["The Desk"](https://github.com/Jason-Latz/jbl-site/pull/6): an interactive, lamplit 3D homepage where every object is live data and a doorway into the site — a turntable that plays what I'm actually listening to, my real bookshelf, a public guestbook notepad, and one persistent world-vs-Jason chess game — with a designed fallback for no-WebGL and reduced-motion visitors.
- On **June 10, 2026**, I ran parallel production-readiness pushes: **Vinyl** 🔒 [PR #15](https://github.com/Jason-Latz/vinyl/pull/15) and [PR #16](https://github.com/Jason-Latz/vinyl/pull/16) swept the app toward App Store submission (the spinning-record motif, recordings that survive auto-lock, privacy-manifest essentials, test suite from 19 to 59), while **Yapper** 🔒 [PR #4](https://github.com/Jason-Latz/yapper/pull/4) and [PR #5](https://github.com/Jason-Latz/yapper/pull/5) hardened the cross-stack contract and then moved the tutor entirely on-device — an Apple FoundationModels → heuristic engine ladder, local persistence, and zero network calls.
- Across **late May 2026**, I kept a set of experiment and optimization tracks running: a new BioBeat repo, a batch of private codex CLI prototypes in my Experiments repo 🔒 (retrieval-lens, score-snapshot, and brief-rendering scripts), and a string of performance PRs across the iOS and web projects.
